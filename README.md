## 🚀 **Getting Started**

### **Prerequisites**
- Node.js 18+
- Python 3.9+
- PostgreSQL 14+
- Redis 6+

### **Installation**

\`\`\`bash
# Clone the repository
git clone https://github.com/syed-amjad65/aog-neural-ai-system.git
cd aog-neural-ai-system

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Run development server
npm run dev
\`\`\`

### **Database Setup**

\`\`\`bash
# Run database migrations
npm run db:migrate

# Seed with sample data
npm run db:seed
\`\`\`

### **AI Models Setup**

\`\`\`bash
# Install Python dependencies
pip install -r requirements.txt

# Train neural models
python scripts/train_models.py

# Start AI services
python scripts/start_ai_services.py
\`\`\`
